# Fall Detection System - Test Suite

This directory contains unit tests for the fall detection system.

## Running Tests

```bash
# Run all tests
pytest

# Run with coverage
pytest --cov=src --cov-report=html

# Run specific test file
pytest tests/test_models.py

# Run with verbose output
pytest -v
```

## Test Structure

- `test_data.py`: Tests for data loading and preprocessing
- `test_models.py`: Tests for model implementations
- `test_metrics.py`: Tests for evaluation metrics
- `test_utils.py`: Tests for utility functions

## Test Data

Tests use synthetic data generated by the `FallDetectionDataset` class to ensure reproducibility and avoid dependency on external datasets.
